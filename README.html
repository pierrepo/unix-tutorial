Trucs et astuces#

Copier / coller#

Pour copier / coller entre Windows et le shell Linux :

  • Pour copier depuis Windows (Ctrl+C) puis coller dans le shell : clic droit de la souris.

  • Pour copier depuis le shell (Ctrl+Maj+C) puis coller dans Windows (Ctrl+V)

Répertoire utilisateur Windows et Unix (dans WSL)#

Depuis un shell Linux, votre répertoire utilisateur de Windows est accessible via le chemin

/mnt/c/Users/omics

omics désigne ici votre login Windows sur les machines du DU.

Nous vous demandons de travailler depuis ce répertoire afin que vos fichiers puissent également être visibles depuis Windows.

De l’aide 🆘#

Si vous avez besoin d’aide, interrogez votre moteur de recherche favori. Une bonne partie des réponses qui vous seront proposées proviendront du site stackoverflow qui est le forum de discussion en informatique.

Les deux tutoriels ci-dessus vous invitent à consulter le manuel (commande man) quand vous avez un doute sur le fonctionnement d’une commande. La documentation du manuel est quasi-exhaustive mais peu conviviale. Le site TLDR pages propose une aide concise sur les commandes Unix et leurs principales options. Par exemple, pour la commande ls.

Enfin, le site explainshell explique le rôle des différentes options utilisées dans une commande. Par exemple pour décompresser une archive avec la commande tar.
